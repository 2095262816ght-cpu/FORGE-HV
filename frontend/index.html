03数据结构说明
数据集构成（论文第 2 章）：
• 样本数：149 条实测合金数据
• 每条样本 = 70 维微结构特征（DINO-v2 自监督模型提取 → PCA 降维） + 22 维成分特征（Al/W/Ta/Ti/Cr/Ni/Mo/Hf/C/Co/B/V/Si/Fe/Nb/Zr/Re/Cb/Ce/Mn/S/P） + 1 维目标值（维氏硬度 HV）
• GAN 扩充数据：10000 条生成样本（论文 5.4 节用）
04预处理流程（论文第 2 章）
1
DINO-v2 微结构特征提取
基于 Vision Transformer 架构的自监督视觉模型 DINOv2-Base，对每张微结构图片提取 768 维视觉特征。对比 clip_b16/b32/l14、dino_b/g/l/s、Nasa 等模型后选用 DINOv2（性能、速度、通用性平衡最佳）。
输入：微结构图 · 输出：768 维向量
2
PCA 主成分分析降维
对 768 维 DINOv2 特征做主成分分析（PCA），抽取前 70 个主成分作为微结构关键特征，去除冗余信息同时保留主要方差。
输入：768 维 · 输出：70 维微结构特征
3
分位数异常值截断（公式 1-3）
采用分位数截断策略保留 1%~99% 分位数范围内的有效数据，将超出上下边界的极端值拉回边界，消除异常值干扰。
下限 Q₁ = 1% · 上限 Q₉₉ = 99%
4
Z-Score 标准化（公式 4-6）
对输入特征和目标变量分别做标准化，转换为均值为 0、标准差为 1 的标准正态分布，消除量级量纲差异，提升训练稳定性。
x' = (x − μ) / σ
5
数据集划分
划分为训练集、验证集、测试集三部分。训练集用于参数学习，验证集用于超参数调优与训练过程监控，测试集用于最终性能评估。

第 3 章 · 网络模型设计（论文图 1）

DDPG 模型架构

深度确定性策略梯度（DDPG）基于 Actor-Critic 架构，适用于连续动作空间回归。状态 = 70 微结构 + 22 成分特征，动作 = 预测 HV，奖励 = 误差驱动反馈。
A架构总览（论文 3.1 节）
核心思想：将硬度预测建模为强化学习环境
• State（状态）：70 维微结构特征 + 22 维成分特征 = 92 维向量
• Action（动作）：预测的维氏硬度值（1 维连续）
• Reward（奖励）：基于预测误差的分段反馈
• Actor：策略网络，根据状态生成动作
• Critic：价值网络，评估动作质量
• 目标网络：Actor/Critic 各配一个，软更新同步参数（公式 7）

D关键机制说明
动作空间（论文 3.3 节）：动作 = 预测的硬度值。Actor 输出经 Tanh 约束到 [-1, 1]，再通过动作缩放机制（公式 9-10）映射到真实 HV 范围。

奖励函数（论文 3.4 节）：五段式奖励结构，根据绝对误差区间给予差异化权重（B=5/3/2/1），叠加相对误差奖励与硬截断，平衡精度与稳定性。

探索策略（论文 4.1 节）：训练期添加高斯噪声（σ₀=0.2，衰减率 0.995，σ_min=0.05），评估期关闭噪声。采用目标策略平滑技巧防止 Critic 过拟合。

优先经验回放（论文 4.2 节）：五元组 (s,a,r,s',done) 存入环形缓冲区，按 TD 误差优先级采样，配重要性采样权重修正偏差（公式 17-19）。

CGAN 架构说明
生成器 G
• 输入：随机噪声 z（高斯分布）
• 网络：多层全连接 + LeakyReLU
• 输出：合成样本（92 维特征 + HV）
• 目标：骗过判别器，生成逼真数据
判别器 D
• 输入：真实样本 or 生成样本
• 网络：多层全连接 + LeakyReLU + Sigmoid
• 输出：真假概率 [0, 1]
• 目标：区分真实数据与生成数据
对抗目标：
minG maxD V(D, G) = Ex~pdata[log D(x)] + Ez~pz[log(1 − D(G(z)))]
训练达到纳什均衡时，G 生成的样本分布逼近真实数据分布
